# 🚀 Rede de Postos de Combustíveis Autônomos - Vale do Paraíba

**Projeto de Logística | Power BI + Python**

Primeiro projeto do meu portfólio. O objetivo é desenvolver um case pioneiro para uma rede de **postos de combustíveis autônomos** (sem frentista) na região do Vale do Paraíba/SP.

### 🎯 Objetivo do Projeto
- Identificar a melhor localização para novos postos autônomos
- Otimizar a logística de abastecimento
- Criar dashboards interativos para suporte à decisão

### 📍 Região de Estudo
- Vale do Paraíba (SP)
- 204 postos de combustíveis existentes (dados ANP)

### 🛠️ Tecnologias Utilizadas
- **Power BI** (análises visuais e mapas)
- **Python** (limpeza e preparação de dados)
- **VS Code + GitHub** (versionamento e documentação)
